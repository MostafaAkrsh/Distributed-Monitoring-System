Notes:
*System developed using Eclipse IDE
*System consist of 4 programs

Steps:
1- Open All the 4 workspace in different 4 instances of Eclipse IDE
2- Run The Server Program first.
3- Run The Computer Program.
3- Run The Sensors Program multible times because the Server need to have multiple inputs before giving the correct output.
4- Run The Driver Program. You may need to run it multiple times.
5- You can now see the required Mobile Agent transfered from server to the driver through Computer.
